css 属性选择器

默认

                                    .demo a{
                                        float:left;
                                        display: block;
                                        height: 50px;
                                        width: 50px;
                                        border-radius: 10px;
                                        background: #79c7c7;
                                        text-align: center;
                                        color: gainsboro;
                                        text-decoration: none;
                                        margin-right: 5px;
                                        font: bold 20px/50px Arial;
                                    }

常用属性选择器：

                            = 
                            *=
                            ^=
                            $=

存在id属性的元素

                                        a[id]{
                                background: yellow;
                            }

1、el[attr=val] 选择 attr 为 val 的 el 元素

                                    a[href = md]{
                                       background: wheat;
                                   }


                                    或者id的first的元素
                                    a[id=first]{
                                        background: #66945e;
                                    }

2、el[attr *= val] 只要el 元素的 attr 属性值里含有 val 就能被选择

                                    a[class*="links"]{
                                       background: yellow;
                                   }

3、el[attr ~= val] el元素中的 attr 属性的属性值 val 单独存在 或 存在的属性值与其他属性值之间用空格隔开就能被选择

                                    a[href ~=abc]{
                                           background: wheat;
                                       }

4、el[attr ^= val] el 元素的 attr 属性值里含有 val 且 val 在最前面才能被选择

                                    a[href^=http]{
                                            background: #132456;
                                        }

5、el[attr $= val] el 元素的 attr 属性值里含有 val 且 在最后面

                                    a[href$=doc]{
                                           background: wheat;
                                       }

6、el[attr |= val] el 元素中 attr 属性的属性值单独存在或属性值之间用 “-” 隔开且属性值 val 在 “-” 前

                                 a[href |=abc-2]{
                                       background: wheat;
                                   }

最常用的有 = *= ^= $= ,其他比较少用
