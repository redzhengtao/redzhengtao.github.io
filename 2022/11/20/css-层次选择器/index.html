css 层次选择器

1、后代选择器

        body p{
        background: #79c7c7;
      }
      在某个元素的后面

2、子选择器

        body >p{
        background: #79c7c7;
      }
      一代 儿子

3、相邻兄弟选择器

        .green + h1{
          background: #66945e;
      }
        指下一个标签

4、通用选择器

         .green ~ h1{
          background: #66945e;
      }
      它的功能是将后续所有的兄弟（相同等级的）标签背景全部改为green色：
