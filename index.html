css 属性选择器

默认

                                    .demo a{
                                        float:left;
                                        display: block;
                                        height: 50px;
                                        width: 50px;
                                        border-radius: 10px;
                                        background: #79c7c7;
                                        text-align: center;
                                        color: gainsboro;
                                        text-decoration: none;
                                        margin-right: 5px;
                                        font: bold 20px/50px Arial;
                                    }

常用属性选择器：

                            = 
                            *=
                            ^=
                            $=

存在id属性的元素

                                        a[id]{
                                background: yellow;
                            }

1、el[attr=val] 选择 attr 为 val 的 el 元素

                                    a[href = md]{
                                       background: wheat;
                                   }


                                    或者id的first的元素
                                    a[id=first]{
                                        background: #66945e;
                                    }

2、el[attr *= val] 只要el 元素的 attr 属性值里含有 val 就能被选择

                                    a[class*="links"]{
                                       background: yellow;
                                   }

3、el[attr ~= val] el元素中的 attr 属性的属性值 val 单独存在 或 存在的属性值与其他属性值之间用空格隔开就能被选择

                                    a[href ~=abc]{
                                           background: wheat;
                                       }

4、el[attr ^= val] el 元素的 attr 属性值里含有 val 且 val 在最前面才能被选择

                                    a[href^=http]{
                                            background: #132456;
                                        }

5、el[attr $= val] el 元素的 attr 属性值里含有 val 且 在最后面

                                    a[href$=doc]{
                                           background: wheat;
                                       }

6、el[attr |= val] el 元素中 attr 属性的属性值单独存在或属性值之间用 “-” 隔开且属性值 val 在 “-” 前

                                 a[href |=abc-2]{
                                       background: wheat;
                                   }

最常用的有 = *= ^= $= ,其他比较少用

css 结构伪类选择器

结构伪类选择器主要根据文档结构来选择器元素，常用于根据父级选择器里面的子元素

选择符 解析
E:first-child 匹配父元素中的第一个子元素 E
E:first-child 匹配父元素中最后一个E元素
E:nth-child(n) 匹配父元素中的第n个子元素E
E:first-of-type 指定类型E的第一个
E:last-of-type 指定类型E的最后一个
E:nth-of-type(n) 指定类型E的第n个 
        ul li:first-child{
        background: #66945e;
    }


        ul li:last-child{
        background: #c75a5a;
    }


         p:nth-child(2){
        background: blue;
    }


         p:nth-of-type(1){
        background: #c75a5a;
    }


        同理也有
            ul li:first-of-type{
        background: #e5e39d;
    }
        和
            ul li:last-of-type{
        background: #e5e39d;
    }
        这里就不放图了

css 层次选择器

1、后代选择器

        body p{
        background: #79c7c7;
      }
      在某个元素的后面

2、子选择器

        body >p{
        background: #79c7c7;
      }
      一代 儿子

3、相邻兄弟选择器

        .green + h1{
          background: #66945e;
      }
        指下一个标签

4、通用选择器

         .green ~ h1{
          background: #66945e;
      }
      它的功能是将后续所有的兄弟（相同等级的）标签背景全部改为green色：
