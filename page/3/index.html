css 结构伪类选择器

结构伪类选择器主要根据文档结构来选择器元素，常用于根据父级选择器里面的子元素

选择符 解析
E:first-child 匹配父元素中的第一个子元素 E
E:first-child 匹配父元素中最后一个E元素
E:nth-child(n) 匹配父元素中的第n个子元素E
E:first-of-type 指定类型E的第一个
E:last-of-type 指定类型E的最后一个
E:nth-of-type(n) 指定类型E的第n个 
        ul li:first-child{
        background: #66945e;
    }


        ul li:last-child{
        background: #c75a5a;
    }


         p:nth-child(2){
        background: blue;
    }


         p:nth-of-type(1){
        background: #c75a5a;
    }


        同理也有
            ul li:first-of-type{
        background: #e5e39d;
    }
        和
            ul li:last-of-type{
        background: #e5e39d;
    }
        这里就不放图了

css 层次选择器

1、后代选择器

        body p{
        background: #79c7c7;
      }
      在某个元素的后面

2、子选择器

        body >p{
        background: #79c7c7;
      }
      一代 儿子

3、相邻兄弟选择器

        .green + h1{
          background: #66945e;
      }
        指下一个标签

4、通用选择器

         .green ~ h1{
          background: #66945e;
      }
      它的功能是将后续所有的兄弟（相同等级的）标签背景全部改为green色：
